Main Endpoints

All the endpoints can be found under the /api/v1/ url.

Testing Endpoints

As the name suggests this is for testing and pinging the api if it is online. The only endpoint is /api/v1/test. it is a simple GET endpoint that is used for testing if the server is currently running.

URL Endpoints

There is only one implemented endpoint here currently. It is a POST endpoint found at /api/v1/url. To use it, use a json object with content like the example below. 

            
                { "url": "https://www.somesite.com" }

Once inputed, it will return an object status code and time of request.
